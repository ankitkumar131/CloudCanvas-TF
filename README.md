# CloudCanvas-TF

Angular workspace scaffold for the CloudCanvas-TF visual GCP-to-Terraform project.

## Required toolchain
- Angular CLI: 21.1.4
- Node.js: 24.13.1

## Quick start
```bash
npm install
npm run start
```

## Current scaffold
- Angular workspace configuration (`angular.json`, strict `tsconfig*`).
- Minimal standalone bootstrap app (`src/main.ts`).
- Feature-first project structure under `src/app/*` matching `SYSTEM_DESIGN.md`.
